Authenticator Symfony permettant de sécuriser une API via un token.

Cette classe implémente un système d'authentification basé sur un token API transmis dans les en-têtes HTTP (header X-AUTH-TOKEN).

Elle permet de :

  • Vérifier la présence du token dans la requête HTTP
  • Authentifier un utilisateur en recherchant son token en base de données
  • Autoriser l'accès si le token est valide
  • Refuser l'accès avec une réponse JSON en cas d'échec

L'authentification s'appuie sur Doctrine pour récupérer l'utilisateur correspondant au token fourni.

Fonctionnement :

  • supports() : vérifie si la requête contient un token
  • authenticate() : valide le token et charge l'utilisateur
  • onAuthenticationSuccess() : laisse passer la requête
  • onAuthenticationFailure() : retourne une erreur JSON (401)
