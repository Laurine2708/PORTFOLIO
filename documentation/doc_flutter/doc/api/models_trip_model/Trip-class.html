Trip

Trip class

Modèle représentant un voyage dans l’application.

Un voyage correspond à un séjour dans une ville donnée et peut contenir plusieurs activités planifiées par l’utilisateur.

Cette classe permet également la sérialisation et la désérialisation des données afin de faciliter les échanges avec une API ou une base de données.
