Récupère une liste de suggestions de lieux à partir d’une chaîne de recherche.

Cette fonction interroge l’API Google Places Autocomplete afin de proposer des lieux correspondant à la saisie de l’utilisateur.

query Texte utilisé pour rechercher des suggestions de lieux.

Retourne une liste de Place contenant la description et l’identifiant du lieu proposé.

En cas d’erreur HTTP, une liste vide est retournée. Les exceptions sont retransmises via rethrow.

Implementation

Future<List<Place>> getAutocompleteSuggestions(String query) async {
  try {
    var response = await http.get(_queryAutocompleteBuilder(query));
    if (response.statusCode == 200) {
      var body = json.decode(response.body);
      return (body['predictions'] as List)
          .map(
            (suggestion) => Place(
              description: suggestion['description'] ?? '',
              placeId: suggestion['place_id'] ?? '',
            ),
          )
          .toList();
    } else {
      return [];
    }
  } catch (e) {
    rethrow;
  }
}
