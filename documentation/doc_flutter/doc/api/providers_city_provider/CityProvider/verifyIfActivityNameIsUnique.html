Vérifie si le nom d’une activité est unique dans une ville donnée.

cityName Nom de la ville concernée. activityName Nom de l’activité à vérifier.

Une requête est envoyée au backend afin de vérifier si une activité portant ce nom existe déjà.

Retourne :

  • null si le nom est unique
  • une réponse JSON contenant l’erreur sinon.

Implementation

Future<dynamic> verifyIfActivityNameIsUnique(
  String cityName,
  String activityName,
) async {
  try {
    City city = getCityByName(cityName);
    http.Response response = await http.get(
      Uri.http(host, '/api/city/${city.id}/activities/verify/$activityName'),
    );
    if (response.statusCode != 200) {
      return json.decode(response.body);
    } else {
      return null;
    }
  } catch (e) {
    rethrow;
  }
}
